Breaking down ownership commands 

`chmod 755 hello_world`


## The order of the numbers
* 7
	* owner of the file
* 5
	* the group
* 5 
	* everyone else


The number break down:
*  7 
	* read, write, and execute
*  6 
	* read, write
*  5 
	* read and execute
*  4 
	* read only
*  3 
	* write and execute 
*  2 
	* write only
*  1 
	* execute only
*  0 
	* no permissions
